taurus
======

This is a Proof of Concept for the taurus app. Quick and dirty
implementation in Dart.

Installation
============

``pub get``

Usage
=====

``dart taurus.dart -h``

Build Snapshot
==============

| ``dart --snapshot=taurus.snapshot taurus.dart``
| ``dart taurus.snapshot -h``
