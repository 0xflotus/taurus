taurux
======

This is a Proof of Concept for the taurux app. Quick and dirty
implementation in Dart.

Installation
============

``pub get``

Usage
=====

``dart src/taurux.dart -h``

Build Snapshot
==============

| ``dart --snapshot=taurux.snapshot src/taurux.dart``
| ``dart taurux.snapshot -h``

Test
====

| ``dart run test test/*``
